# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-09

### Added
- Initial release of Chromatin Priming Index (CPI) methodology
- Analysis of GSE167232 (BAL macrophages, 10,357 cells)
- Analysis of GSE287288 (DTB PBMC, 21,000 cells)
- Integration with 10x Genomics PBMC multiome (1.28M peak-gene links)
- Master regulator TF analysis (55 TFs identified, 16 shared)
- Manuscript for Frontiers in Immunology submission
- GitHub Actions CI/CD pipeline
- Complete documentation and reproducibility files

### Key Results
- CPI range: 77.6% - 89.0% across cell types
- Dendritic cells show highest CPI (89.0%)
- 16 conserved TFs: CEBPB, IRF7, STAT1, JUN, REL, SPI1, RUNX1, ELF1, CEBPD, JUND, IRF3, IRF8, IRF1, XBP1, CEBPA, IRF2
